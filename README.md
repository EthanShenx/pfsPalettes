# pfsPalettes

A lightweight macOS palette floater that stays above other windows and lets you click swatches to copy hex codes.

## Build

1. Open `pfsPalettes.xcodeproj` in Xcode 15+.
2. Select the `pfsPalettes` target.
3. Build and run (⌘R).

## Usage

- Toggle visibility: ⌘⇧C
- Click a swatch to copy its hex value.
- Add colors via hex input, Color Picker, or drag from other apps.
- Import/Export: use the menu button (•••) in the header.

## Requirements

- macOS 13.0 (Ventura) or newer.
